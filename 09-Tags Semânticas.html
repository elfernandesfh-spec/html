Tags semânticas em html

As tags semânticas são elementos HTML que descrevem claramente o significado do conteúdo que elas envolvem. Elas ajudam a melhorar a acessibilidade,a SEO e a estrutura do documento. Aqui estão algumas das principais tags semânticas:

  • <header>: Define o cabeçalho de um documento ou seção.
  • <nav>:Define um conjunto de Links de navegação.
  • <main>: Define a conteúdo principal do documento.
  • <section>:Define uma seção genérica de um documento.
  • <article>: Define um conteudo independente e auto-contido,como um artigo de blog ou nóticia.
  • <aside>:Define um coteudo relacionado,mas se separado do conteudo principal,como uma barra lateral.
  • <footer>:define o rodapé de um documento ou seção.

    • Usar essas tags semânticas ajuda os motores de busca a entender melhor o conteúdo do seu site e melhora aexperiência do usuário, especialmente para aqueles que utilizam Leitores de tela.
